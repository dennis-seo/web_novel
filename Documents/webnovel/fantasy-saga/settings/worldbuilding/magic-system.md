# 마법 체계 -- 에르다스의 마법

---

## 마법의 본질

에르다스의 모든 존재는 그랑카인이 세계에 남긴 잔류 마력 -- **"에테르(Aether)"** -- 속에서 살아간다. 마법이란 시술자의 정신력(마나)으로 이 에테르를 공명시켜 물리적 현상을 일으키는 기술이다.

마법은 단순한 기술이 아니라 **세계와의 대화**에 가깝다. 마법사가 주문을 외우는 것은 세계에 "요청"하는 행위이며, 시술자의 마나와 의지가 강할수록 세계는 더 큰 현상으로 응답한다.

---

## 마법 10단계의 세계관적 의미

마법의 10단계는 단순한 위력 등급이 아니라, **마법사의 존재 자체가 변화하는 과정**을 나타낸다.

### 단계별 의미

| 단계 | 명칭 | 세계관적 의미 | 마법사의 변화 |
|---|---|---|---|
| **1단계** | 입문 (Initiation) | 에테르를 처음 감지하고 조작하는 단계 | 마나 회로가 열림. "세계가 보인다" |
| **2단계** | 초급 (Foundation) | 에테르의 흐름을 읽고 반응하는 단계 | 마나 회로가 안정. 지속적 시전 가능 |
| **3단계** | 중급 (Ascendance) | 에테르를 의지대로 왜곡하는 단계 | 마나 용량 1차 확장. 범위 마법 가능 |
| **4단계** | 상급 (Dominion) | 에테르의 흐름을 지배하기 시작 | 마법서 필요. 고대어 주문 체계 진입 |
| **5단계** | 고급 (Mastery) | 세계의 법칙을 부분적으로 간섭 | 마나 용량 2차 확장. "대가"의 개념 체감 |
| **6단계** | 전문 (Authority) | 특정 영역(원소/신성/저주)의 전문가 | 성향이 확고해짐. 전문 분야 확립 |
| **7단계** | 대마법 (Grandeur) | 세계에 물리적 흔적을 남기는 마법 | 마나 구조 근본 변화. "인간의 한계" 진입 |
| **8단계** | 상위 (Transcendence) | 생사/시공간을 간섭하는 영역 | 수명 연장. 마법이 존재의 일부가 됨 |
| **9단계** | 금역 (Forbidden) | 세계의 법칙 자체를 일시적으로 재정의 | 시전 시 세계에 균열/반작용 발생 |
| **10단계** | 전설 (Legend) | 세계를 변형하는 궁극의 마법 | 세계 자체와 대등한 힘. 대가도 극한 |

### 핵심 규칙: 단계와 대가

- **1~3단계**: 대가가 미미. 마나 소모 = 가벼운 피로감
- **4~6단계**: 대가가 체감됨. 고위 마법 연속 시전 시 코피/실신/마력 고갈
- **7~8단계**: 대가가 위험. 시전 실패 시 마나 역류로 내상/의식 혼탁
- **9단계**: 대가가 극심. 시전 후 며칠간 마법 사용 불가/신체 약화
- **10단계**: 대가가 세계적. 지형 변화, 기후 이상, 마력 파동이 광범위하게 영향

## 마법 성향 체계

에르다스의 마법은 세 가지 성향으로 나뉜다. 이것은 마법의 "선악"이 아니라 **"세계와 교감하는 방식"**의 차이다.

### 로우풀 (Lawful) -- 질서의 마법

- **계열**: 신성/치유/보호/정화
- **원천**: 세계의 질서를 유지하려는 힘과 공명
- **학습처**: 신전, 대사원, 세레니아 엘프 자치령
- **특징**: 안정적이고 예측 가능. 언데드/마족에 강력한 효과
- **제약**: 어둠/저주 계열 마법과 양립 불가. 카오틱 마법 사용 시 로우풀 능력 약화
- **대표 마법**: 힐, 턴 언데드, 홀리 웨폰, 선 버스트, 리절렉션

### 뉴트럴 (Neutral) -- 중립의 마법

- **계열**: 원소(불/물/바람/대지)/유틸리티/강화
- **원천**: 세계의 원소 에너지와 균형적으로 교감
- **학습처**: 마법 아카데미, 독학, 원소 정령
- **특징**: 가장 범용적. 어떤 성향과도 병행 가능
- **제약**: 극한의 위력은 로우풀이나 카오틱에 밀림 (범용의 대가)
- **대표 마법**: 에너지 볼트, 파이어볼, 블리자드, 메테오 스트라이크

### 카오틱 (Chaotic) -- 혼돈의 마법

- **계열**: 저주/사역/흡수/변이/네크로맨시
- **원천**: 세계의 균열과 혼돈에서 힘을 끌어냄
- **학습처**: 금서고, 암흑 조직, 다크엘프 학당, 독자적 연구
- **특징**: 강력하지만 위험. 대가가 크고 중독성 있음
- **제약**: 사용할수록 정신/육체에 침식. "어둠에 물든다"는 표현이 비유가 아님
- **대표 마법**: 칠 터치, 뱀파이어릭 터치, 크리에이트 좀비, 커스 계열

### 성향의 도덕적 딜레마

- 카오틱 마법은 "금지"되었지만 완전히 사라지지 않았다
- 위기 상황에서 카오틱 마법만이 해법일 때 -- "금지된 힘을 쓸 것인가?"
- 카오틱 마법을 사용한 후의 후유증과 주변의 시선
- 다크엘프 사회에서 카오틱은 "자연스러운 것" -- 문화적 상대주의
- **서사적 활용**: 주인공이 카오틱 마법의 유혹을 받는 에피소드 (특히 시즌2)

## 마법 습득 경로

### 1단계~3단계: 정규 학습

- **아르카누스 마법 아카데미**: 엘디아의 정규 교육 기관. 입학~졸업까지 3~5년
- **신전 수습**: 로우풀 계열 마법 특화. 사제 견습으로 학습
- **개인 사사**: 숙련 마법사에게 직접 배움. 지방에서 흔한 경로
- **엘프 자치령 교류**: 정령 마법 특화. 엘프의 허가 필요

### 4단계~6단계: 마법서 기반

- **고대 마법서 필요**: 4단계 이상 마법은 구전이나 단순 학습으로 습득 불가
- **마법서 입수 경로**: 던전 탐험, 유적 발굴, 경매, 암시장
- **학습 조건**: 해당 단계에 도달한 마나 구조 + 기초 마법 숙련도
- **서사적 의미**: 마법서 획득 = 파워업 이벤트. 희소성이 드라마를 만듦

### 7단계~9단계: 대사의 경지

- **극히 희귀한 마법서 + 각성**: 마법서만으로는 부족. 마법사 자신의 "깨달음"이 필요
- **수련의 위험**: 7단계 이상 수련 중 실패하면 마나 역류로 폐인이 될 수 있음
- **대사(Great Mage)**: 7단계 이상을 안정적으로 시전하는 마법사. 대륙에 몇 명 수준
- **서사적 의미**: 대사 급의 마법사와의 조우 = 세계의 거물과 만남

### 10단계: 전설의 영역

- **전설의 마법서**: 세계에 몇 권만 존재. 발견 자체가 역사적 사건
- **자각(Awakening)**: 마법서를 읽는 것이 아니라, 마법서와 "공명"하여 세계의 진실에 닿는 경험
- **습득 = 시즌 클라이맥스**: 10단계 마법 습득과 사용은 각 시즌의 최고 절정
- **세계에 흔적**: 10단계 마법 시전 장소에는 마력의 흔적이 영구히 남음

## 시스템과 마법의 관계

### 마법 습득 시 시스템 알림

새로운 마법을 처음 성공적으로 시전하면 시스템 알림이 발생한다:

```
▶ [스킬 습득] 새로운 마법을 습득했습니다.
   ◆ 파이어볼 (4단계)
```

이것은 "세계가 인정했다"는 의미로, 이후부터 해당 마법의 시전이 안정적으로 가능해진다. 알림 이전에도 시전은 가능하지만 불안정하고 실패 위험이 높다.

### 마법서의 시스템적 의미

마법서는 단순한 교재가 아니라, **고대 마법사의 마나 구조가 기록된 매체**이다. 마법서를 읽으면 시스템이 마법사의 마나 회로에 해당 마법의 "경로"를 새기는데, 이것이 체감으로는 "이해했다"는 깨달음으로 느껴진다.

### 레벨과 마법의 관계

레벨은 마법사의 전반적 마나 용량과 신체 강화를 나타내지만, 마법 습득은 레벨만으로 결정되지 않는다. 레벨이 높아도 마법서 없이는 고위 마법을 배울 수 없고, 마법서가 있어도 마나 구조가 미숙하면 습득에 실패한다.

## 마법 관련 금기

1. **네크로맨시**: 사자를 일으키는 것은 세계의 순환을 거스르는 행위. 공식적으로 금지
2. **영혼 구속**: 타인의 영혼을 속박하여 사역하는 것. 최악의 금기
3. **봉인 해제**: 사대룡이나 마왕급 존재의 봉인을 의도적으로 해제하는 것
4. **시스템 간섭**: 시스템 자체를 조작하려는 시도 (가능한지조차 불명)
5. **대량 학살 마법**: 9단계 이상 마법을 도시/민간인에게 사용하는 것 -- 역사적으로 "멸망의 마법사"라 불린 선례 존재
