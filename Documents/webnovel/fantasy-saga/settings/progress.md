## 프로젝트 개요

- **장르**: 게임 시스템 + 순수 판타지 융합 (D&D풍 중세)
- **규모**: 3시즌 x 70화 = 총 210화+ (시즌4 확장 가능, 열린 결말)
- **에피소드**: 5~7분 분량 (3,500~5,000자)
- **세계명**: 에르다스 (Erdas)
- **주인공**: 카이렌 아셀 (마법사, 협력형)
- **코어 파티**: 카이렌 + 가렌 벨크로스(기사) + 리에나 에란실(엘프 정령술사)
- **로맨스**: 카이렌 ↔ 리에나 (20~30%, 슬로우번)
- **마법**: 리니지1 기반 10단계 (스킬/몬스터만 참조, 스텟/아이템 X)
- **소재**: 은→언데드 특효, 미스릴→상위 언데드/마족 특효
- **시스템**: 게임적 구조 내재 + 시스템 알림 절제 사용. 칭호 없음.
- **드랍**: 몬스터 드랍 특수 아이템 → 일시적 파워업 메커니즘
- **사대룡**: 시즌1~3 복선만, 시즌4+에서 등장
