# 관계도

---

## 코어 파티 내부 관계

### 시각적 관계도

```
              [카이렌 아셀]
             마법사 / 주인공
            /                \
     전우·대조적 성격        지적 동반자 → 연인
     (긴장→신뢰→형제)       (경계→신뢰→사랑)
          /                      \
[가렌 벨크로스]  ——————————  [리에나 에란실]
  기사 / 전위       보호자·투박한 다정       정령술사 / 힐러
                   (불편→이해→가족)
```

### 카이렌 ↔ 가렌: "등을 맡길 수 있는 유일한 존재"

#### 관계 변천
| 시기 | 카이렌 → 가렌 | 가렌 → 카이렌 |
|---|---|---|
| 첫 만남 | "무모한 기사. 전략 없이 돌진하다니" | "책만 읽는 겁쟁이. 칼을 들어야 할 때도 뒤에 있어" |
| 시즌1 초반 | "인정하기 싫지만 저 방패 없으면 난 3분도 못 버텨" | "인정하기 싫지만 저놈 마법 없으면 골렘 한 마리도 못 잡아" |
| 비홀더전 후 | "이 사람은 무모한 게 아니라 용감한 거였어" | "이 녀석은 겁쟁이가 아니라 신중한 거였어" |
| 시즌1 후반 | 말 없이도 전투 호흡이 맞음 | 등을 보여줄 수 있는 유일한 사람 |
| 시즌2 갈등 | "복수에 눈이 멀었어. 너답지 않아" | "네가 뭘 아냐! 가족을 잃어본 적 없잖아" |
| 시즌2 화해 | "네 아픔을 내가 다 이해하진 못해. 하지만 함께 감당할 수 있어" | "...미안. 그리고 고마워" |
| 시즌3 | "가렌은 나의 방패이자 칼" | "카이렌은 나의 지도이자 빛" |

#### 관계의 핵심
- **상보성**: 직감 vs 분석, 행동 vs 사고, 외향 vs 내향. 모든 면에서 반대이기에 완벽하게 보완.
- **갈등 포인트**: 시즌2에서 가렌의 복수심이 폭발할 때 유일하게 카이렌만이 정면으로 맞설 수 있음.
- **독자 관점**: "이 둘의 우정"을 응원하게 되는 브로맨스. 최종전에서 서로를 지키는 장면은 로맨스만큼 감동적.

### 카이렌 ↔ 리에나: "유한한 시간을 함께 채워가는 사이"

#### 관계 변천
| 시기 | 카이렌 → 리에나 | 리에나 → 카이렌 |
|---|---|---|
| 첫 만남 | "무서운 엘프. 정확하고 냉정하다" | "어리석은 인간. 하지만 분석력이 독특하다" |
| 시즌1 초반 | "마법 이론을 이렇게 깊이 나눌 수 있는 사람은 처음" | "이 인간의 질문은... 흥미롭다" |
| 던전 갇힌 밤 | "웃는 얼굴이 이렇게 좋은지 몰랐다" | "이 사람과 있으면 숲에서보다 편하다" |
| 바포메트전 | 무의식적으로 리에나의 안전을 가장 먼저 확인 | "당신이 죽으면 어쩌려고요!" (감정 폭발) |
| 시즌2 갈등 | "왜 혼자 아파하면서 말을 안 해?" | "말해서 뭐가 달라지나요" |
| 시즌2 확인 | "네가 아프면 나도 아파" | "...바보야" (첫 반말) |
| 시즌3 완성 | "50년 뒤에도 네 옆에 있을 거야" | "당신과의 시간으로 채우고 싶어" |

#### 관계의 핵심
- **지적 공명**: 마법 이론 토론이 감정 교류의 시작점. 머리로 시작해서 가슴으로 도달.
- **수명 차이라는 근원적 비극성**: 엘프(수백 년)와 인간(수십 년)의 관계가 주는 절절함.
- **상호 구원**: 카이렌은 리에나에게 "감정을 느껴도 괜찮다"를, 리에나는 카이렌에게 "혼자 감당하지 않아도 된다"를 가르침.

### 가렌 ↔ 리에나: "오빠와 여동생 같은 전우"

#### 관계 변천
| 시기 | 가렌 → 리에나 | 리에나 → 가렌 |
|---|---|---|
| 첫 만남 | "차갑다. 좀 웃어라" | "시끄러운 인간" |
| 시즌1 초반 | "이 엘프, 은근 실력 있잖아" | "무모하지만... 용감한 인간" |
| 비홀더전 후 | "다치면 안 돼. 귀찮으니까 (진짜 이유: 걱정)" | "다치지 마세요. (진심)" |
| 시즌1 후반 | 무조건적 보호 본능 | 투박한 다정함에 점차 마음을 열기 시작 |
| 시즌2 | "내가 앞에 서면 되잖아" (리에나 포함 모두를 지키려) | 가렌의 부상을 치료하며 눈물을 참음 |
| 시즌3 | "카이렌이 못 지키면 내가 지킨다" (리에나를 카이렌만큼 소중히) | "가렌 씨는 나의 첫 번째 인간 가족이에요" |

#### 관계의 핵심
- **로맨스가 아닌 유대**: 독자가 삼각관계를 걱정하지 않도록, 가렌-리에나 관계는 일관되게 "가족적" 유대.
- **보호와 신뢰**: 가렌은 리에나를 무조건 지키려 하고, 리에나는 가렌을 무조건 치유. 무언의 계약.
- **카이렌-리에나 로맨스의 조력자**: 가렌은 두 사람의 감정을 가장 먼저 눈치채고, 은근히 상황을 만들어줌 (일찍 잠들기, 자리 비워주기 등).

## 코어 파티 ↔ 시즌별 게스트 관계

### 시즌1

#### 테오 마르케스
| 관계 | 양상 |
|---|---|
| 테오 → 카이렌 | "카이렌 형! 형이 제일 믿음직해!" (동경 + 의지) |
| 카이렌 → 테오 | 동생 같은 존재. 테오의 밝음이 파티에 필요했음을 이별 후 깨달음 |
| 테오 → 가렌 | "가렌 형은 진짜 멋있어!" (순수한 존경) |
| 가렌 → 테오 | 기사단 시절 잃은 후배를 투영. 지키고 싶지만 붙잡지 않음 |
| 테오 → 리에나 | "엘프 누나... 좀 무서워요" (경외 반 두려움 반) |
| 리에나 → 테오 | "시끄러운 아이... 하지만 정이 가네요" |

#### 엘시아 노르딘
| 관계 | 양상 |
|---|---|
| 엘시아 → 카이렌 | "머리 좋은 꼬마. 전투 감각만 기르면 대단해질 거야" |
| 카이렌 → 엘시아 | 전투의 선배. 마나 관리를 가르쳐준 은인 |
| 엘시아 → 가렌 | "기사 출신? 기본기는 있네. 근데 너무 무리해" |
| 가렌 → 엘시아 | 기사단 이후 처음 존경한 전사. "강해져라"는 말이 성장의 화두 |
| 엘시아 → 리에나 | "넌 강하지만 너무 혼자야. 그건 약함이야" |
| 리에나 → 엘시아 | 같은 "혼자 감당하는 타입"으로서의 공감. 말하지 않지만 이해함 |

### 시즌2

#### 룬델 크라긴
| 관계 | 양상 |
|---|---|
| 룬델 → 카이렌 | "마법사의 무기 취급이 형편없군. 내가 제대로 만들어주마" |
| 카이렌 → 룬델 | 장인에 대한 경외. 미스릴 지팡이를 받은 순간의 감격 |
| 룬델 → 가렌 | "기사 검법은 좋은데 검이 형편없어. 자, 이 미스릴 검을 써봐라" |
| 가렌 → 룬델 | "이 드워프 아저씨, 입은 험하지만 손은 신의 손이야" |
| 룬델 → 리에나 | "엘프라... 흥, 쇠 모르는 나무 족속" (종족 편견) |
| 리에나 → 룬델 | "드워프라... 흥, 나무 모르는 쇠 족속" (응수) |
| **화해 이벤트** | 리에나의 정령석을 룬델이 가공하면서 서로의 기술을 인정 → "나쁘지 않군, 엘프" / "나쁘지 않네요, 드워프" |

#### 카엘 시르반
| 관계 | 양상 |
|---|---|
| 카엘 → 카이렌 | "뉴트럴 마법사? 재미있어. 카오틱의 유혹을 느껴본 적 있나?" |
| 카이렌 → 카엘 | 경계하지만 지적 호기심. 카오틱 마법의 정보가 필요함 |
| 카엘 → 가렌 | "순수한 기사라... 이런 타입은 먼저 죽더군" (비꼼이지만 실은 걱정) |
| 가렌 → 카엘 | "뒤통수 안 칠 거지?" → 시간이 지나며 "이 녀석은 믿을 만해" |
| 카엘 → 리에나 | "순혈 엘프가 인간들과? 재밌는 선택이야, 사촌" (다크엘프는 엘프를 '사촌'이라 부름) |
| 리에나 → 카엘 | "우리가 적이어야 할 이유는 없어요" → 종족 편견 극복의 상징 |

### 시즌3

#### 아이리스 플레인
| 관계 | 양상 |
|---|---|
| 아이리스 → 카이렌 | "카이렌 님은 따뜻한 분이시네요" (순수한 존경, 로맨스가 아닌 가족적 신뢰) |
| 카이렌 → 아이리스 | "아이리스는 도구가 아니야. 네 의지로 여기 있는 거야" |
| 아이리스 → 가렌 | "가렌 님은 오빠 같아요" (아버지를 모르는 아이리스에게 가렌은 보호자 이미지) |
| 가렌 → 아이리스 | 보호 본능 극대화. "네가 원하면 우리 파티에 있어. 아무도 너를 도구로 안 써" |
| 아이리스 → 리에나 | "리에나 님은... 저 같은 하프엘프에게도 다정하시네요" |
| 리에나 → 아이리스 | "당신은 충분히 엘프이고, 충분히 인간이고, 충분히 당신" (리에나의 성장 발언) |

## 코어 파티 ↔ 적대 세력 관계

### 카이렌 ↔ 빌런들

```
[마르쿠스] ← 아카데미 선후배 → [카이렌] ← 가치관 대립 → [세라핀]
                                  ↑
                            카오틱 마법의 유혹
                                  |
                              [발터자르]
```

| 빌런 | 관계 유형 | 핵심 |
|---|---|---|
| 마르쿠스 | 선배 → 적 → (가능: 조력자) | "내가 될 수 있었던 미래" |
| 발터자르 | 유혹자 | "어둠의 길로 빠진 가능성" |
| 세라핀 | 이념적 적 | "설득해야 하는 적. 물리적으로 이기는 것이 아닌 가치의 대결" |

### 가렌 ↔ 빌런들

| 빌런 | 관계 유형 | 핵심 |
|---|---|---|
| 마르쿠스 | 간접 적대 | 마르쿠스의 사교가 가렌 마을 주변을 위협 |
| 발터자르 | 개인적 원한 | 데스나이트 = 아버지 세대의 기사단 → 가렌의 개인적 복수심 |
| 세라핀 | 이념적 적대 | "지키는 자" vs "해방하는 자"의 대립 |

### 리에나 ↔ 빌런들

| 빌런 | 관계 유형 | 핵심 |
|---|---|---|
| 마르쿠스 | 학술적 경멸 | "봉인의 에너지를 전용하겠다고? 어리석은 인간" |
| 발터자르 | 언데드에 대한 엘프적 혐오 | 자연의 순환을 거스르는 존재에 대한 근본적 거부감 |
| 세라핀 | 복잡한 감정 | 세라핀이 리에나에게 "수명 차이"를 무기로 사용 → 리에나의 가장 약한 곳을 찌름 |

## 시즌별 관계 변화 흐름

### 시즌1: 파티의 탄생

```
[개인들의 모임] ————→ [서로를 아는 파티] ————→ [생사를 넘긴 동료]

핵심 전환점:
- 비홀더전: "같이 싸우면 이길 수 있다"
- 바포메트전: "이 사람들 없으면 안 된다"
- 테오/엘시아 이별: "함께하는 것과 떠나보내는 것 모두가 성장"
```

### 시즌2: 파티의 시련

```
[신뢰하는 동료] ————→ [가치관 충돌] ————→ [더 깊은 유대]

핵심 전환점:
- 가렌의 복수심: "신뢰는 갈등을 견뎌낼 때 진짜가 된다"
- 리에나의 비밀 발각: "숨기는 것은 보호가 아니라 단절"
- 데스나이트전: "서로의 약점을 알기에, 더 단단하게 보완한다"
```

### 시즌3: 파티의 완성

```
[단단한 유대] ————→ [세계적 위기의 무게] ————→ [가족]

핵심 전환점:
- 게스트 재등장: "혼자가 아니었다는 것의 증명"
- 세라핀의 유혹: "우리의 답은 파괴가 아닌 공존"
- 최종전: "홀리 라이트닝 스톰(극한 해방) → 기절 → 파티가 지킴 = 함께라서 넘을 수 있는 한계"
```

## 관계 긴장 맵 (갈등 가능성)

| 관계 | 잠재적 갈등 | 발화 시기 | 해소 |
|---|---|---|---|
| 카이렌 ↔ 가렌 | 전략 vs 직감 | 시즌1 전체 | 서로의 방식을 인정 |
| 카이렌 ↔ 가렌 | 복수 vs 냉정한 판단 | 시즌2 중반 | 감정적 대립 → 화해 |
| 카이렌 ↔ 리에나 | 생명력 치유 비밀 | 시즌2 전반 | 대안 마법 개발로 해결 |
| 카이렌 ↔ 리에나 | 수명 차이에 대한 두려움 | 시즌3 중반 | 서약으로 수용 |
| 가렌 ↔ 리에나 | 없음 (이 둘은 갈등보다 이해) | - | - |
| 룬델 ↔ 리에나 | 종족 편견 | 시즌2 초반 | 정령석 가공 협업으로 화해 |
| 카엘 ↔ 리에나 | 엘프/다크엘프 역사적 갈등 | 시즌2 중반 | 개인으로서 서로를 인정 |
| 파티 ↔ 아이리스 | 아이리스의 자기 비하 | 시즌3 전반 | 파티가 아이리스를 "사람"으로 대함 |

## 감정의 흐름 요약

### 카이렌의 인간관계 성장
```
[혼자서 충분하다] → [혼자서는 부족하다] → [함께라서 강하다] → [함께이기에 의미가 있다]
   시즌1 초         시즌1 중반           시즌2              시즌3
```

### 가렌의 인간관계 성장
```
[지키지 못했다] → [지키고 싶다] → [함께 싸우고 싶다] → [함께 살아남겠다]
  과거 트라우마     시즌1           시즌2               시즌3
```

### 리에나의 인간관계 성장
```
[관찰자이고 싶다] → [당사자가 되어버렸다] → [당사자이고 싶다] → [이 사람들의 곁에 있겠다]
   시즌1 초           시즌1 후반            시즌2              시즌3
```
