# 긴장감 곡선 -- 3시즌 에피소드별 텐션 배분

> **시리즈**: 아르카나 연대기
> **세계**: 에르다스 (W.E. 1247년~)
> **총 에피소드**: 210화 (70화 x 3시즌)
> **기준일**: 2026-03-15

## 긴장감 설계 원칙

### 핵심 철학
긴장감은 "지속적인 고조"가 아니라 **파도의 리듬**으로 설계한다. 높은 텐션 뒤에는 반드시 이완 구간이 따르며, 이완 구간에서 독자는 캐릭터의 내면과 관계에 몰입한다. 정점은 한 시즌에 2~3회로 제한하여 희소성을 확보하고, 나머지 구간은 "고조 -> 피크 -> 하강"의 소파도를 반복하며 독자를 끌고 간다.

### 리듬 규칙
1. **전투 연속 제한**: 전투 에피소드 3회 연속 금지. 반드시 탐험/일상/훈련/정보 에피소드를 삽입
2. **이완 배치**: 시즌당 일상/교감 에피소드 3~4회. 텐션 레벨 5(절정) 직후 반드시 텐션 1~2 에피소드 배치
3. **로맨스 간격**: 로맨스 비트 사이 최소 3화 간격. 같은 비트 유형 연속 사용 금지
4. **클리프행어 필수**: 모든 에피소드 마지막에 다음 화에 대한 궁금증/긴장 유발 장치 삽입
5. **3막 구조 엄수**: 각 시즌의 도입(~EP015), 전개(~EP055), 클라이맥스(~EP070)에서 텐션 기저선이 단계적으로 상승
6. **드랍 아이템 소진 = 텐션 상승**: 부스트 아이템 효과 소진 직후 위기/전투 에피소드 배치로 절박감 극대화

### 에피소드 유형 정의
| 유형 | 설명 | 기본 텐션 범위 |
|---|---|---|
| 전투 | 몬스터/적 세력과의 직접 교전 | 3~5 |
| 탐험 | 던전 진입, 지역 탐사, 이동, 퀘스트 수주 | 2~3 |
| 일상/교감 | 파티 내 유대, 일상적 에피소드, 유머, 휴식 | 1~2 |
| 훈련 | 마법 수련, 전술 개발, 장비 제작 | 2~3 |
| 로맨스 | 카이렌-리에나 핵심 감정 비트 (모험 통합형) | 1~4 |
| 정보/설정 | 세계관 정보 공개, 복선 삽입, NPC 대화 | 2~3 |
| 위기/좌절 | 패배, 이별, 내부 갈등, 좌절 | 4~5 |

| **파워업 순간** | 새 마법/장비 획득 직후의 기대감 | 3~4 | 시스템 알림 "마법 각성 -- 4단계" |
| **이별/상실** | 파티원의 이별 또는 중요한 상실 | 3~4 | "형들은 앞으로 가" |
| **복선 투하** | 미래 전개를 암시하는 정보 공개 | 2~3 | "어둠의 기사가 이미 눈을 떴다" |
| **로맨스 여운** | 감정적 순간 뒤의 여운 | 1~3 | "나중에 알려드리겠습니다" |
| **캐릭터 전환** | 새로운 캐릭터 등장/변화 암시 | 2~3 | 그림자 속에서 지켜보는 인물의 실루엣 |
| **상황 전환** | 장소/상황이 크게 바뀌는 순간 | 2~3 | 항구에서 배에 오르는 뒷모습 |

### 클리프행어 운용 규칙

1. **같은 유형 3회 연속 금지**: "전투 예고 -> 전투 예고 -> 전투 예고"는 독자가 패턴을 읽어 긴장감 감소
2. **텐션 5 에피소드의 클리프행어**: 반드시 "위기 고조" 또는 "감정 폭발" 유형 사용
3. **텐션 1 에피소드의 클리프행어**: "로맨스 여운" 또는 "복선 투하"로 부드럽게 다음 화 견인
4. **시즌 마지막 화(EP070)**: "상황 전환" 유형 필수 -- 다음 시즌에 대한 기대감
5. **전투 3연속 구간의 클리프행어**: 중간 화는 "위기 고조", 마지막 화는 "파워업 순간" 또는 "감정 폭발"로 해소감 부여
