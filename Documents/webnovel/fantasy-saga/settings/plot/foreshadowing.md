# 복선/회수 추적표

> **시리즈**: 아르카나 연대기
> **총 에피소드**: 210화 (70화 x 3시즌)
> **추적 대상**: 세계관 / 캐릭터 / 플롯 / 로맨스 복선
> **최종 갱신**: 시즌3 종료 시점 기준

---

## 복선 관리 원칙

### 심기/회수 규칙

| 규칙 | 설명 |
|---|---|
| **3화 규칙** | 심어진 복선은 최소 3화 이후에 힌트를 제공해야 한다. 즉각 회수는 복선이 아닌 설정 설명 |
| **시즌 내 회수 우선** | 단기 복선(P 카테고리)은 원칙적으로 심은 시즌 내에 회수. 시즌을 넘기면 중기 이상으로 승격 |
| **미회수 한도** | 한 시즌에서 열어놓을 수 있는 미회수 복선은 최대 8개. 초과 시 독자의 기억 부하 발생 |
| **힌트 간격** | 장기 복선(W 카테고리)은 10~20화 간격으로 힌트를 공급. 독자가 "잊힐 만하면" 상기시키는 리듬 |
| **회수 시 감정 비트** | 복선 회수는 반드시 캐릭터의 감정 반응을 동반. 설정 정보만 전달하면 카타르시스 부재 |
| **중복 심기** | 하나의 복선을 2개 이상의 채널(대사, 배경, 아이템)로 심어 독자에게 자연스럽게 각인 |

### 카테고리 분류

| 카테고리 | 코드 | 회수 주기 | 특징 |
|---|---|---|---|
| 세계관 복선 | W | 장기 (시즌 단위, S4+ 연결 포함) | 사대룡, 봉인, 시스템, 그랑카인 등 세계의 근본 미스터리 |
| 캐릭터 복선 | C | 중기 (10~40화) | 인물의 과거, 내면 갈등, 성장 전제조건 |
| 플롯 복선 | P | 단기~중기 (5~30화) | 빌런 정체, 전투 열쇠, 장비 단서, 사건 전개 |
| 로맨스 복선 | R | 시즌 단위 | 카이렌-리에나의 감정 진전, 엘프어, 정령 반응 |

### 상태 코드

| 상태 | 의미 |
|---|---|
| **완료** | 심기-힌트-회수가 모두 완결됨 |
| **부분 회수** | 일부 회수되었으나 잔여 요소가 남아 있음 |
| **미완 (S4 연결)** | S3 종료 시점에서 의도적으로 열어둠. S4+ 소재 |
| **진행 중** | 심기 완료, 힌트 공급 중, 회수 대기 |
