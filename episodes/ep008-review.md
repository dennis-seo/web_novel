### 수정 필요 사항

1. [긴급] 제로 코드 흑요석 내부 통신 감청 -- 권한 레벨 초과 능력
   - **위치**: 마지막 시스템 UI (`[흑요석 길드 -- 내부 통신 감청 단편]`)
   - **문제**: 권한 레벨 2의 원본 감정은 기본적으로 몬스터/아이템/공간의 숨겨진 정보를 읽는 수준이다. magic-system.md에 따르면 "타인의 시스템 코드 부분 열람"은 권한 레벨 61~80 구간이다. 길드 내부 통신 감청은 인물 정보 심층 열람보다 더 침습적인 기능. EP007에서 인물 심층 열람조차 권한 레벨 10이 필요하다고 했는데, 통신 감청은 그보다 높은 수준이어야 함.
   - **수정 제안**: UI의 성격을 변경. "내부 통신 감청"이 아니라 "원본 감정이 현장의 잔여 마력 서명과 사건 결과를 분석해 추론한 결과"로 처리. 구체적으로:
     ```
     ╔════════════════════════════════════════════════════╗
     ║  // ZERO CODE v0.0 -- EVENT TRACE ANALYSIS         ║
     ╠════════════════════════════════════════════════════╣
     ║                                                    ║
     ║  [사건 사후 추적 분석]                             ║
     ║                                                    ║
     ║  김재훈 소속 확인: 흑요석 길드                     ║
     ║  길드장: 이주혁 (S급)                              ║
     ║                                                    ║
     ║  흑요석 길드 행동 패턴 분석 (원본 DB):             ║
     ║  "소속원 체포 시 현장 전원 신원 조회"              ║
     ║  발동 확률: 98.7%                                  ║
     ║                                                    ║
     ║  ⚠ 조회 대상에 당신이 포함될 가능성이             ║
     ║    높습니다.                                       ║
     ║                                                    ║
     ╚════════════════════════════════════════════════════╝
     ```
     이렇게 하면 직접 감청이 아닌 '패턴 분석'이 되어 권한 레벨 2에서도 가능한 범위 내로 조정됨. 한시우의 전생 지식과도 결합 가능.

2. [권장] 분량 압축 -- 목표 범위 초과
   - **위치**: 보스 전투 장면 (약 8줄)
   - **문제**: 보스 전투는 이 에피소드의 핵심이 아님(밀수 사건이 핵심). 보스 전투 묘사가 "C급 전사 박태환이 정면을 잡고~" 부분에서 약 8줄을 차지하는데, 3~4줄로 압축 가능.
   - **수정 제안**: "보스 전투는 순조로웠다. 15분 만에 클리어. 한시우의 관심은 보스 마석이 아니었다." 정도로 축약.

3. [권장] 백도현 전화 장면의 배치 강화
   - **위치**: "그게 더 무서운 거야." 직후 전화 종료
   - **문제**: "그게 더 무서운 거야"가 EP008에서 가장 인상적인 대사 중 하나인데, 전화가 바로 끊기고 이주혁 UI로 넘어감. 이 대사의 여운이 짧음.
   - **수정 제안**: 전화 종료 후 한시우의 1문장 반응 추가. "한시우는 어둠 속에서 한참 휴대폰을 들고 있었다." 정도로 대사의 무게를 느끼게 한 뒤 이주혁 UI로 전환.

4. [권장] 한시우의 가방 투척 -- 자연스러움 보강
   - **위치**: "짐 가방 하나를 들어올렸다. 무겁다. 포터용 장비 가방. 20킬로가 넘는다."
   - **문제**: 20킬로 가방을 15미터 거리의 종유석을 정확히 맞힐 수 있는가? STR 18(F급)으로는 무거운 가방을 그 거리/정확도로 투척하기 어려움.
   - **수정 제안**: 거리를 7~8미터로 줄이거나, 가방이 아닌 가방 속 가벼운 장비 1개를 꺼내 던지는 것으로 변경. 또는 가방을 굴려서 종유석 기둥 하단에 부딪히게 하는 방식. 포터의 짐 운반 기술(정확한 위치에 짐을 내려놓는 기술)과 연결하면 캐릭터성이 강화됨.

### 잘된 점

- 한시우의 문제 해결 방식이 일관됨: 직접 행동하지 않고, 다른 사람이 발견하도록 유도. 이 패턴이 EP004의 보스 약점 알려주기, F17 복선(서아가 발견하게 유도)과 완벽히 연결
- "사실이었다. 거짓말은 한 글자도 없었다." -> "그게 더 무서운 거야." -- 백도현의 직감이 한시우의 은밀한 행동을 관통하는 장면. 두 사람의 관계 깊이를 보여줌
- 김재훈의 1초 흔들림 -> 박태환도 봄 -> "이미 늦었다" -- 미세한 심리전을 시각적으로 처리한 좋은 서술
- 전투 후 최영수의 "미, 미친 거 아냐?" 에 대한 한시우의 "괜찮아요. 끝났으니까." -- 이 짧은 대화가 일반인과 한시우의 심리적 거리를 보여줌
- "1막이 끝나고 있었다" -- 메타 서술이지만 한시우의 자기 인식으로 읽히는 이중 기능. 독자에게 구조적 전환을 알리면서도 캐릭터의 관조적 성격을 반영
