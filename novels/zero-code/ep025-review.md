### 수정 필요 사항
1. **시스템 UI 박스 폭**: 경고 메시지 UI의 긴 문장들('원본 감정 사용 횟수: 허용 기준 182% 초과' 등)이 모바일에서 줄바꿈 문제를 일으킬 수 있음. 줄 내부 텍스트를 짧게 분리하거나 박스 폭을 축소할 것.
2. **문체 미세 조정**: 213행 "시스템이 스스로 띄운 메시지"에서 '스스로'는 정확한 표기이나, 시스템을 의인화하는 맥락에서 "시스템이 자동으로 띄운 메시지" 또는 "시스템이 자발적으로 띄운 메시지"가 더 정확한 뉘앙스를 전달할 수 있음. 선택적 수정.

### 잘된 점
- '세 개의 그림자' 구조가 강현준(위)/이주혁(아래)/관리자(시스템) 삼면포위를 직관적으로 전달한다.
- 서아의 "모르는 척은 해줄게요"가 캐릭터 본질(직접적이지만 존중하는 태도)을 정확히 드러냄.
- 백도현, 오달수, 서아의 "혼자 하지 마라" 변주가 자연스럽게 연결됨.
- 마지막 장면에서 한시우가 '일어서는' 물리적 행동이 내면의 결단과 겹치는 연출이 좋음.

### 복선 체크
- **F07 "혼자 하지 마라" 반복**: 170~171행에서 백도현/오달수/서아의 같은 맥락 발언이 명시적으로 연결됨. 3막 내 EP028 회수를 위한 포석 정상 배치.
- **F24 관리자 "즉각 조치"**: 228~230행 "권한 레벨 상한 접근 시 즉각 조치 발동 예정" 명시. EP027 디버그 트리거로 이어질 준비 완료.
- **F05 강현준 카메라 꺼진 후 표정**: 본 에피소드에서는 직접 등장하지 않으나, 내부 보고서 작성 언급으로 강현준의 독자적 추적 동기가 강화됨. 정상 진행.
- 본 에피소드에서 회수 예정인 복선은 없으며, 3막 도입부로서 복선 포석이 적절히 수행됨.
