### 수정 필요 사항
1. **AP 수치 재확인**: 본 에피소드 시작 시 "현재 AP: 77"(271행)이나 EP031에서 코드 수정 후 AP 77이었고, EP032에서 추가 AP 변동 언급이 없었으므로 77 유지는 일관됨. 다만, EP031의 AP 112->77 문제(EP030 종료 시 AP 52였던 것과의 불일치)가 여전히 미해결. **EP030~031 사이의 AP 변동을 소급 수정해야 이 수치가 살아남.**
2. **서아 마나 잔량**: 283행 "마나가 30퍼센트 이하"로 서아의 소모가 극심하나, 이후 던전 코어에서 보스급 몬스터와 추가 전투를 벌임. 마나 30% 미만에서 S급 보스급을 상대하는 것이 가능한지 -- 서아의 각성 등급(A급)을 고려하면 극한의 상황으로 설명 가능하나, "마나 포션 사용" 또는 "기본 검술로 버텼다" 등의 한 줄이 있으면 설득력 보강.

### 잘된 점
- 프롤로그(EP001) 상황 재현이 완벽. "그때도 천장이 무너졌다" / "아무도 돌아보지 않았다" -> "이번에는 혼자가 아니었다"의 대비가 전체 서사의 핵심 주제(혼자에서 함께로)를 완성.
- 트리거 코드가 "인류를 죽이는 것이 아니라 쫓아내는 것"이라는 관리자의 의도 해석이 EP034 관리자 대면을 자연스럽게 준비.
- 백도현의 "다시 오는 거 없어. 한 번이야."가 전생에서도 들었던 말이지만 맥락이 달라진다는 처리가 정교.
- 코드 수정 성공률 67%의 긴장감 + AP 잔량 카운트다운이 몰입도 극대화.

### 복선 체크
- **F09 코드 수정 실패 위험**: 268~270행 "수정 실패 시: 코어 즉시 폭발 -> 브레이크 가속" + 성공률 67% + "코드 수정 실패 시 해당 스킬은 영구 소멸"(368행). 대규모 수정에서의 실패 위험이 구체적으로 구현됨. 회수 완료.
- **F15 서아 속성 전환**: 186행 "속성 전환이 검날 위에서 빛의 색을 바꿨다. 파란색에서 붉은색으로. 붉은색에서 하얀색으로." + 208행 "하얀색에서 검은색으로. 암흑 속성." -- 연속 속성 전환(화염/빙결/암흑)의 풀 가동. EP027 치유 응용과 합산하여 F15 회수 완료.
- **F04 여의도 심층부 삭제**: 트리거 코드 비활성화로 기점 봉인 구조 재편성, 심층부 접근 경로 해제(436~438행). EP034 진입을 위한 최종 문 개방. 회수 진행 중.
- **F22 프롤로그 포터 두고 탈출**: 전생 플래시백에서 "아무도 돌아보지 않았다"(233행)로 프롤로그 상황을 재소환. EP035 에필로그에서 최종 회수 예정.
