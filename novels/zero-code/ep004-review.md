### 수정 필요 사항

1. [권장] 분량 미세 초과 -- 정산 후 출구까지의 전환(162~167행)에서 "포터는 마지막에 나간다"의 설명이 약간 과잉. "파티원들은 먼저 나갔다. 포터는 마지막이다."로 압축 가능.

2. [권장] 16행: "종류를 알 수 있는 사람이었다. 포터를 사람이 아니라 장비 운반 기능으로 보는 종류." -- "종류"가 두 번 반복되어 어색. "한눈에 알 수 있는 유형이었다. 포터를 사람이 아니라 장비 운반 기능으로 보는 유형." 또는 후반부를 별도 문장으로 분리.

3. [권장] 10행 "포터 생활 5년차" -- 회귀 후 시점에서 포터 생활은 아직 2년차 정도. "5년치의 기억을 가진 몸" 또는 "전생 5년의 몸이 기억하고 있었다" 등으로 시제를 명확히 할 것. 현재 문맥에서 "5년차의 몸은 3년 전으로 돌아와도 무게를 기억하고 있었다"로 읽히긴 하지만, 초독자가 혼동할 여지 있음.

4. [권장] 보스 전투 후 최영석이 질문하지 않고 넘어가는 전개(146행)가 자연스러우나, 궁수가 한시우를 가리키는 장면에서 궁수의 반응을 한 줄 추가하면 더 입체적. 예: "궁수가 한시우를 가리켰다. 본인도 왜 포터의 말을 들었는지 약간 당혹스러운 표정이었다."

5. [권장] 169~170행: "원본 감정이 반응했다. 한시우가 발동한 것이 아니었다. 저절로 뜬 것이었다." -- 원본 감정이 자동 발동되는 전례가 설정에 없음. magic-system.md에서 원본 감정은 "사용 시 AP 미세 소모"라고 되어 있으므로, 자동 발동이면 AP가 소모됐는지 여부를 짧게 언급하면 설정 정합성 강화. 예: "시야 구석에 제로 코드 창이 떠올랐다. AP가 0.1 줄어 있었다."

6. [권장] metadata `chars: 5685` -- 실제와 차이. 재측정 권장.

### 잘된 점
- 시스템 UI가 system-ui.md 템플릿을 정확히 따름. EP001~003과 달리 v2.0/v0.0 구분이 시각적으로 명확. 이 포맷을 EP001~003에도 소급 적용해야 함.
- 포터의 현실을 보여주는 방식이 탁월. "물을 받으면서 한시우를 보지 않았다. 손만 뻗었다." -- 한 줄로 계급 관계를 보여줌.
- "6%. 보스의 약점을 찾아낸 사람이 받는 금액이 아니라, 짐을 나른 사람이 받는 금액이었다." -- 구조적 불평등을 행동으로 각인시키는 문장. F22 복선 강화.
- 원본 감정의 실전 활용이 첫 번째로 구체적으로 드러나는 에피소드. "숨겨진 약점" -> "간접 전달" -> "성공"의 흐름이 한시우의 전략적 캐릭터성과 완벽 부합.
- 삭제 영역 감지 클리프행어가 EP005로의 자연스러운 다리 역할.
